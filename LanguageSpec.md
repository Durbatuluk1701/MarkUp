# Language Spec - Written in Markdown

| Input          | Translation            | Description     |
| -------------- | ---------------------- | --------------- |
| \# {Input}     | \<h1>{Input}\</h1>     | Heading 1       |
| \## {Input}    | \<h2>{Input}\</h2>     | Heading 2       |
| \### {Input}   | \<h3>{Input}\</h3>     | Heading 3       |
| \**{Input}**   | \<b>{Input}\</b>       | Bold Text       |
| \*{Input}*     | \<i>{Input}\</i>       | Italicized Text |
